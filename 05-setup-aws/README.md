## Provision AWS Resource with Terraform

### Terraform Commands
1. `Initiate` **Terraform**
```shell
make tf_init
```

2. `Plan` **Terraform**
```shell
make tf_plan
```

3. `Apply` **Terraform**
```shell
make tf_apply
```

4. `Destroy` **Terraform**
```shell
make tf_destroy
```

5. `Clean` **Environment**
```shell
make tf_clean
```
